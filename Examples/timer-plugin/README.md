`Timer` is a sample plugin demonstrating the [Mirabox Space SDK](https://sdk.key123.vip).

# Description

The timer plugin records time on a button press.

# Features

* Written in Vue

* Cross-platform (macOS, Windows)

### Install

```bash
$ cd timer
```

```bash
$ pnpm i
```

### Development

```bash
$ pnpm dev
```

### Format

```bash
$ pnpm format
```

### Build

```bash
$ pnpm build
```
